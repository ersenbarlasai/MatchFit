# MatchFit Agent Documentation

Bu klasör MatchFit agent mimarisinin uygulama ve Antigravity çalışma dökümantasyonudur.

## Tek Ana Kaynak

Kodlama sırasında esas alınacak dosya:

`CANONICAL_AGENT_ARCHITECTURE.md`

Çelişki durumunda bu dosya kazanır.

## Destek Dosyaları

- `matchfit_agent_mimarisi_detayli_dokuman.md`: İnsan için uzun açıklamalı doküman.
- `agent_inventory.md`: Tüm agentların rol, okuma/yazma ve statü dökümü.
- `relationship_matrix.md`: Agentlar arası ana ilişkiler.
- `gap_analysis.md`: Word raporu ve mevcut repo karşılaştırmasına göre eksik/çakışan alanlar.
- `antigravity_structure.md`: Antigravity için kök dosya ve klasör yapısı.
- `implementation_backlog.md`: Kod tarafında uygulanması gereken sonraki adımlar.

Kök `AGENTS.md`, `GEMINI.md`, `.agents` ve `.agent/rules/matchfit-workspace.md` dosyaları canonical dosyaya yönlendiren kısa girişlerdir.
